Online Store

React online store. Navigation is implemented with ReactRouter library. Authentication, registration and database are processed with google Firebase cloud service. User information, products and shopping cart are managed with Redux, Reselect and stored with Redux-Persist. Components are styled with SASS and Styled-components library. Notification component is managed with React Context, which is shown when a new item is added to a cart. The application is deployed on Netlify.

Different version of this project (implemented with various technologies) are shown below on this page. You can access them by checking out the specified branches on my GitHub.
