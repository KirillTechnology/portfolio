Top hacker news

Hacker news website (news.ycombinator.com) posts a lot of very interesting articles about the tech world every day. The problems is there are too many of them and you don't have time to scroll throught all of them.

This Python script analyses first n pages and shows you only the most important articles based on the minumum specified rating.

Password checker

Website (haveibeenpwned.com) has a database of all user data leaks from the most popular websites like Facebook and Twitter. You can check if you login and password exists in this database.

This Python script uses parcial hashes and k-anonymity principle to securily communicate with the website's API and check if your passwords have been leaked.

Offline translator

When you are traveling in a foreign country and you don't know the local language, you cannot always rely on the infinite power of Internet. Sometime it is simply not available.

This Python script uses 'translate' package that can translate your text without Internet. Supported languages: English, Spanish, Portugese and Mandarin.
